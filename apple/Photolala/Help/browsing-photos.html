Browsing Photos

Photolala provides powerful tools for efficiently browsing large photo collections. Learn how to navigate, view, and interact with your photos.

Grid View

The default grid view displays your photos as thumbnails for quick visual scanning.

Adjusting Thumbnail Size

  • Slider Control - Drag the thumbnail size slider in the toolbar
  • Keyboard Shortcuts:
    • ⌘+ (Mac) / Ctrl++ - Increase thumbnail size
    • ⌘- (Mac) / Ctrl+- - Decrease thumbnail size
    • ⌘0 (Mac) / Ctrl+0 - Reset to default size
  • Preset Sizes - Choose from Small, Medium, Large, or Extra Large

Navigating the Grid

Use these methods to move through your photos:

  • Mouse/Trackpad - Scroll vertically through the grid
  • Arrow Keys - Navigate in any direction
  • Page Up/Down - Jump by full pages
  • Home/End - Jump to first or last photo

Preview Mode

Double-click any thumbnail to enter preview mode for detailed viewing.

Thumbnail Strip

The thumbnail strip at the bottom provides quick navigation:

  • Click any thumbnail to jump directly to that photo
  • Current photo is highlighted
  • Scroll horizontally to see more thumbnails
  • Hide/show with ⌘T (Mac) or toolbar button

Selecting Photos

Single Selection

Click once on any thumbnail to select it. The selected photo will be highlighted.

Multiple Selection

  • Range Selection - Click first photo, then Shift+click last photo
  • Individual Selection - ⌘+click (Mac) or Ctrl+click to add/remove photos
  • Select All - ⌘A (Mac) or Ctrl+A
  • Deselect All - Click empty space or press Escape
Tip: Right-click on selected photos to access the context menu with quick actions.

Context Menu

Right-click (or Control+click on Mac) any photo to access quick actions:

  • Open - Open in default application
  • Open With... - Choose specific application
  • Show in Finder/Explorer - Reveal file location
  • Get Info - View detailed metadata
  • Copy - Copy selected files
  • Move to Trash - Delete selected files

Performance Tips

For smooth browsing of large collections:

  • Thumbnail Cache - Photolala automatically caches thumbnails for faster loading
  • Progressive Loading - Thumbnails load as you scroll
  • Memory Management - Older thumbnails are released when browsing large folders
Note: Very large folders (10,000+ photos) may take time to initially scan. The status bar shows loading progress.
