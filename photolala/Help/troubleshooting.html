Troubleshooting

Solutions to common issues you might encounter while using Photolala.

Performance Issues

Slow Thumbnail Loading

Problem: Thumbnails take a long time to appear or load slowly.

Solutions:

  • Wait for initial thumbnail generation to complete
  • Check available disk space for cache storage
  • Clear thumbnail cache: Preferences > Cache > Clear Cache
  • Reduce thumbnail size for faster loading
  • Ensure photos are on a fast storage device (SSD recommended)

Application Freezes or Hangs

Problem: Photolala becomes unresponsive.

Solutions:

  • Check if a large folder is being scanned (see status bar)
  • Reduce number of photos displayed at once
  • Close unused windows to free memory
  • Restart the application
  • Update to the latest version

High Memory Usage

Problem: Photolala uses excessive memory.

Solutions:

  • Close windows with large photo collections
  • Reduce thumbnail cache size in Preferences
  • Enable memory optimization: Preferences > Performance
  • Restart the app periodically when working with many photos

Display Issues

Thumbnails Not Showing

Problem: Blank or missing thumbnails.

Solutions:

  • Check file permissions on photo folders
  • Verify photos are in supported formats
  • Regenerate thumbnails: Right-click > Regenerate Thumbnail
  • Clear and rebuild cache
  • Check for corrupted image files

Wrong Image Orientation

Problem: Photos appear rotated incorrectly.

Solutions:

  • Photolala respects EXIF orientation data
  • Rotate images: Right-click > Rotate
  • Check camera orientation settings
  • Update image metadata if needed

Color Profile Issues

Problem: Colors look different than expected.

Solutions:

  • Enable color management: Preferences > Color
  • Calibrate your display
  • Check embedded color profiles in images
  • Use consistent color space (sRGB recommended)

File Access Problems

Cannot Open Folder

Problem: Error when trying to open a photo folder.

Solutions:

  • Check folder permissions
  • Ensure folder exists and is accessible
  • Verify network connection for network drives
  • Grant Photolala folder access in System Preferences
  • Try accessing parent folder first

Missing Photos

Problem: Some photos don't appear in folder view.

Solutions:

  • Check file extensions are supported
  • Verify hidden files setting: View > Show Hidden Files

  • Check filter settings aren't hiding photos
  • Ensure files aren't corrupted

Search Problems

Search Not Finding Photos

Problem: Search returns no or incorrect results.

Solutions:

  • Wait for indexing to complete (see status bar)
  • Rebuild search index: Preferences > Search > Rebuild Index
  • Check search syntax is correct
  • Clear search filters that may be limiting results
  • Verify metadata is present in photos

Platform-Specific Issues

macOS

App Not Opening

  • Check Gatekeeper settings in System Preferences
  • Right-click app and choose Open first time
  • Verify macOS version compatibility

Permission Dialogs

  • Grant folder access when prompted
  • Check System Preferences > Security & Privacy > Files and Folders
  • Add Photolala to Full Disk Access if needed

iOS/iPadOS

Cannot Access Photos

  • Grant photo library access in Settings
  • Check Settings > Photolala > Photos
  • Select "All Photos" for full access

Storage Full

  • Clear thumbnail cache regularly
  • Offload unused data in Settings
  • Use iCloud Photos for storage

Error Messages

Common Errors and Solutions

Error Cause Solution
"Access Denied" No permission to folder Grant folder access or check permissions
"Invalid Image" Corrupted or unsupported file Verify file integrity or convert format
"Cache Full" Thumbnail cache at limit Clear cache or increase cache size
"Network Timeout" Slow network drive Check network connection or copy locally

Getting More Help

Diagnostic Information

When reporting issues, include:

  • Photolala version (About Photolala)
  • Operating system version
  • Steps to reproduce the problem
  • Error messages (exact text)
  • Sample files if applicable

Support Resources

  • Online Help - Visit our website for latest guides
  • Community Forum - Get help from other users
  • Email Support - support@photolala.app
  • Bug Reports - GitHub issues tracker

Debug Mode

Enable debug logging for detailed diagnostics:

  1. Open Preferences > Advanced
  2. Enable Debug Mode
  3. Reproduce the issue
  4. Send logs with support request
Tip: Most issues can be resolved by updating to the latest version. Check for updates regularly via Photolala > Check for Updates.

Preventive Measures

  • Keep Photolala updated
  • Maintain adequate free disk space
  • Organize photos in manageable folder sizes
  • Back up your photo collection regularly
  • Use supported image formats
  • Clear cache periodically
