Organizing Photos

Keep your photo collection organized with Photolala's sorting, filtering, and management features.

Sorting Options

Arrange your photos in the order that makes most sense for your workflow:

Available Sort Methods

  • Name - Alphabetical order by filename
  • Date Modified - Most recent changes first
  • Date Created - Original creation date
  • Size - File size from largest to smallest
  • Type - Group by file extension

Changing Sort Order

  1. Click the Sort button in the toolbar
  2. Select your preferred sort method
  3. Click again to reverse the order (ascending/descending)
Tip: Your sort preference is remembered per folder, so different folders can have different sort settings.

Filtering Photos

Focus on specific photos by filtering the display:

Filter by Type

  • Images Only - Show only image files (JPEG, PNG, etc.)
  • RAW Only - Display only RAW format files
  • Videos Only - Show video files
  • All Files - Display everything

Filter by Date

Use the date filter to find photos from specific time periods:

  • Today
  • This Week
  • This Month
  • This Year
  • Custom Date Range

Working with Folders

Folder Navigation

Photolala's folder structure mirrors your file system:

  • Subfolders - Double-click to navigate into subfolders
  • Back Button - Return to parent folder
  • Path Bar - Click any folder in the path to jump directly
  • Recent Folders - Quick access to recently viewed folders

Multi-Window Workflow

Open multiple folders simultaneously for comparison or organization:

  1. Use File > Open Folder to open additional windows
  2. Arrange windows side-by-side
  3. Drag photos between windows to reorganize

Batch Operations

Perform actions on multiple photos at once:

Selecting Multiple Photos

  • Select All - ⌘A (Mac) / Ctrl+A
  • Select Range - Click first, Shift+click last
  • Select Individual - ⌘+click (Mac) / Ctrl+click

Common Batch Actions

  • Move - Drag selected photos to new folder
  • Copy - Hold Option (Mac) / Alt while dragging
  • Delete - Move to trash (with confirmation)
  • Export - Save copies with different settings
Caution: Batch operations affect all selected files. Always verify your selection before performing destructive actions.

Metadata and Information

Viewing Photo Information

Access detailed information about your photos:

  • Quick Info - Hover over thumbnail for basic details
  • Info Panel - Press ⌘I (Mac) for full metadata
  • EXIF Data - Camera settings, GPS location, etc.

Metadata Display Options

  • Show/hide filename below thumbnails
  • Display file size and dimensions
  • Show capture date and time
  • Display camera and lens information

Best Practices

Folder Structure

Organize your photos effectively:

  • Date-based - Year/Month/Day hierarchy
  • Event-based - Group by occasions or projects
  • Location-based - Organize by shooting location
  • Hybrid - Combine approaches as needed

Naming Conventions

Use consistent file naming:

  • Include date in filename (YYYY-MM-DD)
  • Add descriptive keywords
  • Use sequential numbering for series
  • Avoid special characters
