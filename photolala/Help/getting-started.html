Getting Started with Photolala

Welcome to Photolala, your powerful cross-platform photo browser. This guide will help you get up and running quickly.

First Launch

On macOS, Photolala opens directly to the photo browser. To start browsing:

  1. Choose File > Open Folder or press ⌘O
  2. Navigate to the folder containing your photos
  3. Click Open to view your photos

On iOS and iPadOS, Photolala presents a welcome screen:

  1. Tap Select Folder on the welcome screen
  2. Navigate to your photo folder
  3. Tap Open to start browsing

Understanding the Interface

The main Photolala interface

Key Interface Elements

  • Thumbnail Grid - Displays your photos as thumbnails
  • Navigation Bar - Shows current folder and navigation controls
  • Status Bar - Displays photo count and selection information
  • Thumbnail Strip - Quick navigation through photos (when previewing)

Basic Navigation

Opening Folders

Photolala uses a window-per-folder approach:

  • Each folder opens in its own window
  • Navigate subfolders within the same window
  • Open multiple folders in separate windows for comparison

Viewing Photos

Double-click any thumbnail to open it in preview mode. In preview mode you can:

  • View photos at full size
  • Navigate with arrow keys
  • Use the thumbnail strip for quick navigation
  • Press Escape to return to grid view
Tip: Hold Shift while clicking to select multiple photos in sequence, or ⌘ (Mac) / Ctrl (Windows) to select individual photos.

Customizing Your View

Adjust how your photos are displayed:

  • Thumbnail Size - Use the slider or menu options to adjust thumbnail size
  • Sort Options - Sort by name, date, or size
  • Show/Hide Details - Toggle file information display
