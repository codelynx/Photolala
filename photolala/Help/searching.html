Searching Photos

Find specific photos quickly using Photolala's powerful search and filtering capabilities.

Quick Search

  2. Type your search terms

What You Can Search

  • Filename - Full or partial file names
  • Extension - File types (jpg, raw, png)
  • Date - Creation or modification dates
  • Metadata - EXIF data, keywords, descriptions
  • Camera Info - Camera model, lens, settings

Advanced Search Syntax

Use special operators for more precise searches:

Operator Example Finds
name: name:sunset Files with "sunset" in the name
ext: ext:raw All RAW format files
date: date:2024 Photos from 2024
camera: camera:nikon Photos taken with Nikon cameras
size: size:>10mb Files larger than 10MB

Combining Search Terms

  • AND - Space between terms (default)
  • OR - Use | between terms
  • NOT - Use - before term
  • Exact phrase - Use quotes

Examples:

  • sunset beach - Contains both "sunset" AND "beach"
  • sunset | sunrise - Contains "sunset" OR "sunrise"
  • vacation -work - Contains "vacation" but NOT "work"
  • "golden hour" - Exact phrase "golden hour"

Smart Filters

Use pre-defined smart filters for common searches:

Built-in Filters

  • Recent - Photos from the last 7 days
  • Favorites - Starred or rated photos
  • Untagged - Photos without keywords
  • Large Files - Photos over 20MB
  • RAW Files - All RAW format images
  • Videos - Video files only

Creating Custom Filters

  1. Perform a search with your desired criteria
  2. Click the Save Filter button
  3. Name your filter
  4. Access it from the Filters menu

Search by Metadata

EXIF Data Search

Search using camera settings and technical data:

  • iso:1600 - Specific ISO value
  • aperture:f/2.8 - Specific aperture
  • shutter:1/500 - Shutter speed
  • focal:50mm - Focal length
  • lens:"24-70mm" - Lens model

Location Search

Find photos by GPS data (if available):

  • location:paris - City name
  • country:france - Country
  • gps:yes - Has GPS coordinates
  • near:"eiffel tower" - Near landmark

Date Range Searches

Find photos from specific time periods:

Date Formats

  • date:2024 - Entire year
  • date:2024-06 - Specific month
  • date:2024-06-15 - Specific day
  • date:today - Today's photos
  • date:yesterday - Yesterday's photos
  • date:week - Last 7 days
  • date:month - Last 30 days

Date Ranges

  • date:2024-01..2024-06 - Range of months
  • date:>2024-01-01 - After specific date
  • date:<2024-01-01 - Before specific date

Search Tips

Pro Tips:
  • Use autocomplete suggestions for faster searches
  • Save frequently used searches as Smart Filters
  • Combine multiple criteria for precise results
  • Use the search history dropdown for recent searches

Search Results

Working with Results

  • Result Count - Shows number of matching photos
  • Highlight - Search terms highlighted in metadata
  • Refine - Add more terms to narrow results
  • Export Results - Save search results as collection

Search Performance

Photolala indexes your photos for fast searching:

  • Initial indexing happens on first folder open
  • Index updates automatically when files change
  • Large folders may take time to fully index
  • Search becomes faster after initial indexing
Note: Complex searches on very large collections may take a moment. The search progress indicator shows indexing status.
